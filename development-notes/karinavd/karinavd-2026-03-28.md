3. A responsive grid has been configured to ensure cards are displayed correctly across different screen resolutions
- Проблемы: 
While developing the dashboard, an architectural question arose: what is the best way to organize the display of a grid of cards? At first, I considered using standard Flexbox with `flex-wrap`, but I found that at certain screen sizes, the cards could behave unpredictably (for example, the last card in the row would stretch to fill the entire width)
- Решения (или попытки): 
To improve layout consistency on the Dashboard, I decided to switch to CSS Grid, which made the layout much more stable. The Card component now has a clean interface and scales easily to accommodate new entities in the project
